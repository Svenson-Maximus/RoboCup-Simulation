# RoboCup-Simulation
This repository showcases a Siemens Plant Simulation digital twin of the RoboCup Logistics League (RCLL), which models a smart manufacturing environment where fleets of autonomous mobile robots collaborate to produce customized products under constraints such as limited time, resource oversubscription, and uncertain execution behavior.

The demo focuses on the RCLL's core challenge: online planning and scheduling in a dynamic production environment. Using a digital twin, different planning, scheduling, and multi-robot coordination strategies can be tested and evaluated without requiring a physical robot fleet. The simulation provides a platform for analyzing system performance, resource utilization, bottlenecks, and production throughput while exploring approaches for reliable long-term autonomous operation and error handling.

Inspired by the RCLL's vision of intelligent factory automation, this project demonstrates how digital twins can support the development and validation of smart manufacturing solutions before deployment in real-world systems.
